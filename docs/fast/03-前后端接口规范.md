# 表格填写助手 —— 前后端接口规范

**文档编号**: LingDoc-FAST-003  
**版本**: v1.0  
**基础路径**: `/lingdoc/form`  
**鉴权**: JWT Bearer Token  

---

## 1. 接口清单

### 1.1 任务管理接口

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| POST | `/lingdoc/form/upload` | 上传表格文件并创建任务 | `lingdoc:form:upload` |
| GET | `/lingdoc/form/list` | 查询任务列表 | `lingdoc:form:list` |
| GET | `/lingdoc/form/{taskId}` | 获取任务详情 | `lingdoc:form:list` |
| DELETE | `/lingdoc/form/{taskId}` | 删除任务 | `lingdoc:form:delete` |
| DELETE | `/lingdoc/form/batch` | 批量删除任务 | `lingdoc:form:delete` |

### 1.2 字段管理接口

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/lingdoc/form/fields/{taskId}` | 获取任务字段列表 | `lingdoc:form:list` |
| PUT | `/lingdoc/form/fields` | 批量更新字段值 | `lingdoc:form:edit` |
| PUT | `/lingdoc/form/fields/confirm` | 批量确认字段 | `lingdoc:form:edit` |

### 1.3 生成与下载接口

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| POST | `/lingdoc/form/generate` | 触发AI生成填写后文档 | `lingdoc:form:generate` |
| GET | `/lingdoc/form/download/{taskId}` | 下载填写后文档 | `lingdoc:form:download` |
| GET | `/lingdoc/form/preview/{taskId}` | 预览文档（原始/填写后） | `lingdoc:form:list` |

### 1.4 参考文档接口

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/lingdoc/form/references/{taskId}` | 获取参考文档列表 | `lingdoc:form:list` |
| POST | `/lingdoc/form/references/select` | 手动选择参考文档 | `lingdoc:form:edit` |

## 2. 接口详情

### 2.1 上传表格文件

```
POST /lingdoc/form/upload
Content-Type: multipart/form-data
Authorization: Bearer {token}
```

#### Request

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `file` | MultipartFile | 是 | 空白表格文档（PDF/DOCX/XLSX） |
| `taskName` | String | 否 | 任务名称，不传则自动从文件名提取 |

#### Response

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
    "taskId": "task_a1b2c3d4e5f6",
    "taskName": "国家奖学金申请表",
    "originalFileName": "国家奖学金申请表.docx",
    "originalFileUrl": "/upload/lingdoc/form/2026/04/19/task_a1b2c3d4e5f6_国家奖学金申请表.docx",
    "status": "1",
    "statusName": "识别中",
    "createTime": "2026-04-19 10:30:00"
  }
}
```

### 2.2 查询任务列表

```
GET /lingdoc/form/list?pageNum=1&pageSize=10&status=&taskName=
Authorization: Bearer {token}
```

#### Request Parameters

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `pageNum` | Integer | 否 | 页码，默认 1 |
| `pageSize` | Integer | 否 | 每页条数，默认 10 |
| `status` | String | 否 | 状态筛选：0/1/2/3/4 |
| `taskName` | String | 否 | 任务名称模糊搜索 |

#### Response


## 3. 前端 API 封装

**文件**: `src/api/lingdoc/form.js`

```javascript
import request from '@/utils/request'

// 上传表格文件
export function uploadForm(data) {
  return request({
    url: '/lingdoc/form/upload',
    method: 'post',
    data: data,
    headers: { 'Content-Type': 'multipart/form-data' },
    timeout: 60000
  })
}

// 查询任务列表
export function listForm(query) {
  return request({
    url: '/lingdoc/form/list',
    method: 'get',
    params: query
  })
}

// 获取任务详情
export function getFormTask(taskId) {
  return request({
    url: '/lingdoc/form/' + taskId,
    method: 'get'
  })
}

// 批量更新字段
export function updateFormFields(data) {
  return request({
    url: '/lingdoc/form/fields',
    method: 'put',
    data: data
  })
}

// 触发AI生成
export function generateForm(data) {
  return request({
    url: '/lingdoc/form/generate',
    method: 'post',
    data: data,
    timeout: 120000
  })
}

// 下载填写后文档
export function downloadForm(taskId) {
  return request({
    url: '/lingdoc/form/download/' + taskId,
    method: 'get',
    responseType: 'blob'
  })
}

// 获取参考文档列表
export function getFormReferences(taskId) {
  return request({
    url: '/lingdoc/form/references/' + taskId,
    method: 'get'
  })
}

// 删除任务
export function delForm(taskId) {
  return request({
    url: '/lingdoc/form/' + taskId,
    method: 'delete'
  })
}
```

## 4. 统一响应格式

所有接口统一返回 `AjaxResult`：

```json
{
  "code": 200,        // 200成功，其他为错误码
  "msg": "操作成功",   // 提示信息
  "data": { }         // 业务数据
}
```

### 错误码定义

| code | 说明 |
|------|------|
| 200 | 成功 |
| 500 | 服务器内部错误 |
| 1001 | 参数校验失败 |
| 1002 | 文件上传失败 |
| 1003 | 文件格式不支持（仅支持 PDF/DOCX/XLSX） |
| 1004 | 任务不存在 |
| 1005 | 无权限操作该任务 |
| 1006 | AI 服务调用失败 |
| 1007 | 文档生成失败 |
| 1008 | 文件下载失败 |
