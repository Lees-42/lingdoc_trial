## 使用方式

1. **前端演示模式**：点击页面「加载演示数据」按钮，直接填充模拟数据，无需后端服务
2. **后端联调测试**：将 HTML 文件作为上传文件，测试 `/lingdoc/form/upload` 接口
3. **截图对比**：将空白表格与 AI 填写后的表格并排对比，验证排版效果

---

## 输出结果示例

AI 填写后，系统应生成如下文件：

```
国家奖学金申请表_已填写_20260419.html
差旅费报销单_已填写_20260419.html
培训报名表_已填写_20260419.html
```

填写后的表格中，所有 `[待填写]` 占位符应被替换为实际值，保留原有表格样式。
